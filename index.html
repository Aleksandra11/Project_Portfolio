Transportation Application

This application allows users to select a departure and arrival train station, and see a list of trains, times, and durations. A default train schedule is provided and should be used when the application is offline. If a network connection exists, the application query an endpoint that provides information about all arrival and departure times. This application is focused on user experience in challenging connection scenarios with poor or no internet connection. The app works both online and offline, and loads the data through cache.
